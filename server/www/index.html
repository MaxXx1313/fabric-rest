What to do next:

  • Go to admin section to manage hyperledger
  • Use docker volume to mount your web app here or just put static files in www folder:
    docker run -v /path/to/your/app:/usr/src/app/www maxxx1313/fabric-rest
  • Use docker volume to mount your middleware here or just put the files in middleware folder:
    docker run -v /path/to/your/middleware:/usr/src/app/middleware maxxx1313/fabric-rest

    The middleware is a function, which should return an express middleware.

    Example

    middleware/sample.js
    module.exports = function(require){
      // use 'require' to load server modules
      return function(req, res, next) {
        console.log('[%s] %s %s', new Date().toISOString(), req.method.toUpperCase(), req.url);
        next();
      };
    }
    middleware/map.json
    {
      "middleware":[
        {
          "name":"Sample module",
          "file":"./sample.js"
        }
      ]
    }
